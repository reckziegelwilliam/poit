# Environment Variables for Vercel

## Required Environment Variables

Copy these to your Vercel project settings (Settings â†’ Environment Variables):

### 1. Vercel Blob Storage (automatically added when you enable Blob)
BLOB_READ_WRITE_TOKEN=(generated by Vercel)

### 2. Security - Phone Number Restriction
ALLOWED_PHONE=+12074237861

### 3. Upload Token (generate a secure random string)
UPLOAD_TOKEN=pk_live_51234567890abcdefghijklmnopqrstuvwxyz_EXAMPLE

### 4. Optional - Site URL (Vercel provides this automatically)
NEXT_PUBLIC_SITE_URL=https://your-project.vercel.app

## How to Generate a Secure Upload Token

Option 1: Use a UUID generator:
https://www.uuidgenerator.net/

Option 2: Use OpenSSL in terminal:
openssl rand -hex 32

Option 3: Use Node.js:
node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"

## Important Notes

- Keep your UPLOAD_TOKEN secret and never commit it to git
- The ALLOWED_PHONE must match exactly what you send from your iOS Shortcut
- Include the country code (+1 for US)
- BLOB_READ_WRITE_TOKEN is automatically provided by Vercel when you enable Blob storage
